With [Upptime](https://upptime.js.org), you can get your own unlimited and free uptime monitor and status page, powered entirely by a GitHub repository. We use [Issues](https://github.com/Tediware/tediware-upptime/issues) as incident reports, [Actions](https://github.com/Tediware/tediware-upptime/actions) as uptime monitors, and [Pages](https://status.tediware.com) for the status page.
